Problem:
Industrial hopper systems suffer from unstable material flow, overflow risks, and lack of precise control over vibration-based feeding mechanisms.

Objective:
Design and implement an automated control system to regulate material flow, ensure stability, and prevent overflow using PLC-based logic.

System Architecture:
Sensor Inputs → PLC Control Logic → Actuator Control (Vibration Motor / Output Control) → Feedback Monitoring

Engineering Implementation:
• Designed electrical schematics using QElectroTech
• Implemented PLC control logic for real-time system regulation
• Structured safety conditions to prevent overflow and system overload
• Integrated feedback-based decision logic for stable operation

Key Engineering Challenges:
• Maintaining stable material flow under variable load conditions
• Avoiding continuous vibration instability

Result:
Achieved stable and controlled material feeding behavior with improved safety against overflow conditions and reduced system instability.

Control Systems | Embedded Logic

Smart Conveyor System: IR-Based Automated Sorting

Problem:
Manual sorting systems in small-scale production lines suffer from inconsistent positioning, delays, and lack of real-time decision control.

System Design:
Designed an embedded control architecture using IR sensors for object detection, combined with logic-gated motor control for real-time actuation of conveyor segments.

Architecture:
Sensor Layer → Signal Detection (IR) → Control Logic (Embedded C / decision rules) → Actuator Layer (DC motor control).

Engineering Challenges:
• Noise and false detection from IR sensors
• Timing synchronization between detection and motor response
• Ensuring stable object positioning under continuous flow

Solution Approach:
Implemented logic filtering to validate sensor triggers and introduced state-based motor control to prevent overlapping operations and misalignment.

Result:
Achieved reliable object detection and stable sorting behavior under continuous operation, improving system consistency and reducing positioning errors in simulation.

Tech Stack:
IR Sensors, Embedded C, DC Motor Control, Logic State Machine Design
